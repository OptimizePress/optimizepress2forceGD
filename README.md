# OptimizePress Ecommerce fixOptimizePress 2 force GD
##Description

This plugin forces GD extension for image manipulation, usefull when content templates reset returns 500 Internal Server Errors

##Installation

1. Click Download ZIP button on the right
2. Install plugin archive through WordPress' Add plugin screen
3. Activate the plugin
